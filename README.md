# Server Discord Bot

Discord 서버 관리와 다양한 유틸리티 기능을 제공하는 멀티 기능 Discord 봇입니다.

## 기능

### 🖥️ 서버 관리
- **서버 추가/제거**: 게임 서버 등록 및 관리
- **서버 시작/종료**: 이모지 반응을 통한 직관적인 서버 제어
- **서버 상태 확인**: 실행 중인 서버 목록 조회
- **서버 업데이트**: 서버 파일 자동 업데이트

### 🎲 게임 유틸리티
- **제비뽑기**: 음성 채널 또는 사용자 목록에서 랜덤 선택
- **팀 나누기**: 자동으로 균등하게 팀을 구성
- **사다리타기**: 참가자와 결과를 매칭하는 사다리게임

### 🔧 기타 기능
- **음성 채널 연동**: 음성 채널 참가자 자동 감지

## 프로젝트 구조

```
src/
├── bot/           # Discord 봇 관련 파일들
│   ├── index.js           # 메인 봇 로직
│   ├── botCommands.js     # 명령어 정의
│   ├── draw.js           # 제비뽑기 기능
│   ├── ladder.js         # 사다리타기 기능
│   ├── teamSplit.js      # 팀 나누기 기능
│   └── webhook.js        # Webhook 서버
├── server/        # 서버 관리 관련 파일들
│   ├── serverMng.js      # 서버 관리 로직
│   ├── start_server.py   # 서버 시작 스크립트
│   ├── quit_and_close.py # 서버 종료 스크립트
│   └── update_server.py  # 서버 업데이트 스크립트
├── utils/         # 유틸리티 함수들
│   └── utils.js          # 공통 유틸리티
└── config/        # 설정 파일들
    ├── .env              # 환경 변수
    └── servers.json      # 서버 설정
```

## 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. 환경 변수 설정
`.env` 파일을 생성하고 Discord 봇 토큰을 설정:
```env
DISCORD_TOKEN=your_discord_bot_token_here
```

### 3. 실행
```bash
# 슬래시 명령어 등록 및 봇 실행 (개발 모드)
npm run dev

# 슬래시 명령어만 등록
npm run deploy-commands

# 봇만 실행
npm start

# PM2를 사용한 프로덕션 실행
pm2 start ecosystem.config.cjs
```

## 명령어

> 💡 **새로운 슬래시 명령어를 사용하세요!** 더 직관적이고 사용하기 쉽습니다.

### 🎯 슬래시 명령어 (권장)

#### 게임 유틸리티
- `/제비뽑기` - 제비뽑기 진행 (음성 채널 선택 UI 제공)
- `/팀나누기` - 팀 나누기 (음성 채널 선택 UI 제공)
- `/사다리 설정` - 사다리타기 참가자 설정
- `/사다리 결과` - 사다리타기 결과 설정
- `/사다리 시작` - 사다리타기 실행

#### 서버 관리
- `/서버 목록` - 등록된 서버 목록 확인
- `/서버 시작` - 서버 선택하여 시작
- `/서버 종료` - 서버 종료
- `/서버 추가` - 새 서버 등록
- `/서버 제거` - 서버 제거
- `/서버 상태` - 실행 중인 서버 확인
- `/서버 업데이트` - 서버 업데이트

#### 기타
- `/도움말` - 모든 명령어 확인

### 🎮 슬래시 명령어 사용법

#### 음성 채널 연동
1. `/제비뽑기` 또는 `/팀나누기` 명령어 입력
2. 당첨자 수 또는 팀 수 입력
3. **나에게만 보이는 UI**에서 음성 채널 선택
4. 선택한 채널의 멤버들로 자동 진행
5. 결과는 **전체 채팅에 공개**

#### 직접 입력
1. 슬래시 명령어에서 "참가자" 옵션 사용
2. 또는 "직접 입력하기" 버튼 클릭
3. 모달 창에서 참가자들을 공백으로 구분하여 입력

#### 특징
- ✅ **Ephemeral 응답**: 명령어 입력 과정은 나에게만 보임
- ✅ **자동완성**: 명령어 입력 시 Discord가 자동으로 제안
- ✅ **직관적 UI**: 드롭다운 메뉴와 버튼으로 쉬운 선택
- ✅ **에러 방지**: 입력값 검증으로 실수 방지

### 📝 기존 텍스트 명령어 (하위 호환성)

### 기본 명령어
- `$명령어` - 사용 가능한 모든 명령어 목록 표시

### 서버 관리
- `$서버추가 [서버명] [경로] [설명]` - 새 서버 추가
- `$서버목록` - 등록된 서버 목록 조회
- `$서버시작` - 이모지를 통한 서버 시작
- `$서버종료 [서버명]` - 서버 종료
- `$서버제거 [서버명]` - 서버 제거
- `$실행서버` - 현재 실행 중인 서버 목록
- `$업데이트 [서버명]` - 서버 업데이트

### 게임 유틸리티
- `$제비뽑기 음성채널 [채널명] [당첨자수]` - 음성 채널에서 제비뽑기
- `$제비뽑기 사용자 [참가자들] [당첨자수]` - 사용자 목록에서 제비뽑기
- `$팀나누기 음성채널 [채널명] [팀수]` - 음성 채널에서 팀 나누기
- `$팀나누기 사용자 [참가자들] [팀수]` - 사용자 목록에서 팀 나누기
- `$사다리 설정 [참가자들]` - 사다리타기 참가자 설정
- `$사다리 결과 [결과들]` - 사다리타기 결과 설정
- `$사다리 시작` - 사다리타기 실행
